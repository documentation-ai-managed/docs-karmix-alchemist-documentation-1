Karmix Alchemist provides an intuitive interface for creating, editing, and organizing documentation. This guide covers the core features and workflows.

## Creating Documentation

<Steps>
  <Step title="Navigate to Project" icon="folder">
    Select your project from the dashboard.
  </Step>
  <Step title="Create Page" icon="file-plus">
    Click "New Page" to start writing.
  </Step>
  <Step title="Edit Content" icon="edit">
    Use the rich text editor or Markdown.
  </Step>
  <Step title="Save and Publish" icon="send">
    Save drafts or publish immediately.
  </Step>
</Steps>

## Collaboration Features

<Callout kind="info">Enable real-time collaboration for better team productivity.</Callout>

| Feature | Description | Access Level |
|---------|-------------|--------------|
| Comments | Add notes and discussions | All users |
| Reviews | Request approvals for changes | Editors+ |
| Sharing | Share links with external users | Configurable |
| Notifications | Get updates on changes | All users |
