## Integration Configuration

<Tabs>
  <Tab title="GitHub" icon="github">
    Sync documentation with GitHub repositories.

    <ParamField path="repo" param-type="string" required="true">
      GitHub repository in format owner/repo.
    </ParamField>

    <ParamField query="branch" param-type="string" required="false">

    </ParamField>
  </Tab>
  <Tab title="Slack" icon="message-square">
    Receive notifications in Slack channels.

    <ParamField header="Authorization" param-type="string" required="true">
      Slack bot token for API access.
    </ParamField>
  </Tab>
  <Tab title="Jira" icon="file-text">
    Link documentation to Jira issues.

    <ParamField path="project" param-type="string" required="true">

    </ParamField>
  </Tab>
</Tabs>

## Advanced Settings

<Callout kind="info">Advanced settings require admin privileges.</Callout>

| Setting | Type | Description |
|---------|------|-------------|
| Version Control | Toggle | Enable/disable automatic versioning |
| Public Access | Toggle | Allow public viewing of documentation |
| API Rate Limits | Number | Set API request limits per hour |
| Webhook URLs | String | Configure webhook endpoints |

<CodeGroup tabs="JSON,YAML">
  ```json
{
  "project": {
    "name": "My Documentation Project",
    "settings": {
      "versionControl": true,
      "publicAccess": false,
      "rateLimit": 1000,
      "webhooks": [
        "https://example.com/webhook"
      ]
    }
  }
}
  ```
  ```yaml
project:
  name: My Documentation Project
  settings:
    versionControl: true
    publicAccess: false
    rateLimit: 1000
    webhooks:
      - https://example.com/webhook
  ```
</CodeGroup>

## Environment Variables

<Expandable title="Required Variables" default-open="false">
  - `KARMIX_TOKEN`: Your API authentication token
  - `KARMIX_BASE_URL`: API base URL (default: https://api.karmixalchemist.com/v1)
  - `KARMIX_PROJECT_ID`: Default project ID for operations
</Expandable>

<Callout kind="tip">Store sensitive configuration values as environment variables for security.</Callout>
